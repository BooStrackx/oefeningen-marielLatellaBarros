Variabelen

In PHP wordt een variabele aangeduid door een dollarteken ($) gevolgd door een aantal geldige symbolen (letters, hoofdletters, cijfers en de underscore (_)). Een variabele mag echter niet met een cijfer beginnen. Variabelen zijn ’case-sensitive’.

In dit hoofdstuk worden variabelen van het type boolean, integer, double, string (char bestaat niet in PHP) en NULL besproken. Meer informatie: PHP.net variables en PHP.net vardump.
Let bij het datatype string vooral op het verschil tussen single en double quotes. Bij single quotes wordt de inhoud letterlijk overgenomen, bij double quotes worden variabelen en worden geëscape’te symbolen zoals \n geëxpandeerd.

Variabelen typen en functie var_dump en gettype

Het type van de variabele wordt bepaald door de context. Het type wordt opgevraagd via de functie gettype en via de functie var_dump wordt de volledige informatie van de 2 variabele afgedrukt.

Cast-operaties

Het is ook mogelijk omzettingen tussen verschillende types te maken via cast-operaties.
De cast-regels naar datatypes zijn terug te vinden in de documentatie:
Boolean cast
Integer cast
Float cast
Juggling cast

De cast naar boolean gebeurt via de operaties (boolean) of (bool). De volgende waarden worden naar false gecast:

  • de integer 0
  • de double 0.0
  • een lege string en de string met waarde "0
  • NULL

Alle andere waarden van het type integer, double of string worden naar true gecast.

De cast naar integer/ float gebeurt via de operaties (integer) of (int). De volgende regels gelden voor deze operatie:

  • true wordt naar 1 gecast, false naar 0
  • een double wordt naar beneden afgerond (12.34 wordt 12)
  • een string die niet begint met een cijfer wordt naar 0 omgezet.
    ("test 123" wordt 0). Een string die wel begint met een aantal cijfers wordt omgezet naar het overeenkomende getal. Spaties in het begin van de string worden hierbij genegeerd. Ook worden symbolen na de reeks cijfers niet in rekening gebracht. (" 123aad" wordt 123).
  • Null wordt 0

De omzetting naar string gebeurt via de cast-operatie (string). Van numerieke waarden wordt eenvoudig een string-representatie gegeven. De boolean true wordt omgezet naar "1" en de boolean false naar een lege string. Ook Null wordt omgezet naar de lege string.

Arrays

In PHP is een array een datatype waarin een aantal waarden (values) bewaard kunnen worden.
Bij elke waarde in de array (value) is er een sleutel (key) voorzien waarmee de waarde opgevraagd kan worden. (In veel programmeertalen wordt deze datastructuur een ’map’ genoemd.) Een array kan aangemaakt worden via de functie array: $a = array(element_1, element_2, ... ,element_N); waarbij element_i = value_i element_i = key_i => value_i (index)

De waarden (values) in een array hoeven niet allemaal van hetzelfde type te zijn: in eenzelfde rij kunnen waarden van verschillende types bewaard worden. De sleutel (key) moet van het datatype integer of string zijn. Als er geen key gesp wordt bij het aanroepen van de functie array dan wordt de eerstvolgende beschikbare integer genomen als key.
